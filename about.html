About TravelBloom

TravelBloom is a simple travel recommendation site built to help people get from “I want to go somewhere” to “here are a few places I’d actually enjoy.” We focus on quick suggestions with a short description and an image, so you can decide fast and keep planning.

This project is a demo for learning front-end web development. The destinations shown are examples, and the “contact” form is for practice (it doesn’t send real messages).
